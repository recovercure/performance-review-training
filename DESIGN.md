# DESIGN.md

## 气质与意象

绩效面谈陪练 Agent 是一款企业级管理培训工具。意象锚点：一间朝南的高管教练室——午后阳光透过百叶窗洒在胡桃木桌面上，空气中弥漫着淡淡咖啡香，教练与学员隔着一张不大不小的圆桌对坐，语气平和但直击要害。不是冰冷的培训系统，而是有温度的成长对话。

关键词：沉稳、专业、聚焦、成长

## 视觉策略

- 摄影/图像方向：不使用照片素材，以几何图形和渐变营造空间感
- 图形语言：圆角卡片 + 细线图标 + 数据可视化图表（雷达图、进度环）
- 员工角色头像：使用几何抽象图形 + 色彩编码（每个角色类型一个主色调）

## 配色方案

| 角色 | 色值 (oklch) | 意象来源 |
|------|-------------|---------|
| 主色-深青 | `oklch(0.45 0.08 200)` | 深秋湖面倒影，沉稳可信 |
| 强调-琥珀金 | `oklch(0.75 0.15 70)` | 午后阳光透过百叶窗的暖光 |
| 背景-暖白 | `oklch(0.99 0.005 80)` | 宣纸般的温润底色 |
| 成功-翠 | `oklch(0.65 0.15 150)` | 春日新叶 |
| 警示-赤陶 | `oklch(0.6 0.18 30)` | 落日余晖 |
| 中性灰 | `oklch(0.55 0.01 200)` | 雾天远山 |

## 字体排版

- 字体族：PingFang SC / Microsoft YaHei（系统已配置）
- 标题：font-weight 600-700，紧凑行高
- 正文：font-weight 400，行高 1.7 增强可读性
- 数字/评分：tabular-nums，等宽对齐

## 动效与交互

- 运动性格：沉稳缓动，不急躁
- 缓动曲线：`cubic-bezier(0.16, 1, 0.3, 1)` —— 快速开始，缓慢落地
- 流式输出：逐字渲染，光标闪烁，模拟真实对话节奏
- 步骤切换：横向滑入，前一步淡出
- 评分展示：数字滚动动画 + 雷达图渐入

## 页面结构

- **首页**：左侧统计面板（练习次数、平均分、技巧提升曲线），右侧快捷入口卡片
- **训练页**：顶部四步进度条，左侧对话区，右侧实时分析侧栏
- **历史页**：列表 + 详情抽屉，支持复盘回放
- **知识库**：左右分栏，左侧文档列表，右侧导入面板

## 设计禁忌

- 不要使用科技蓝/紫色渐变
- 不要使用 emoji 作为功能图标
- 不要使用霓虹光效或过度阴影
- 不要使用全圆角（border-radius 过大），保持商务克制
- 不要在面谈对话区域使用高饱和度背景色
